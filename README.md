# GMEL

A fast-paced 2D arcade experience where you drive and smash everything in your path.

![Game Jam Event](https://img.itch.zone/aW1nLzI2NjY3NzYyLnBuZw==/original/6lQNhF.png)

## Overview
Jump into your car, crash through breakable objects, and watch them explode into satisfying pixel chunks with dynamic effects and camera shake. Switch between player and vehicle to navigate the level and clear all obstacles to progress.

This game was made for a game jam and built using **Godot**, then converted into **HTML** for web play.

## Controls

### Player
- **A / D**: Move left / right
- **SPACE**: Jump

### Car
- **ENTER**: Enter / Exit car
- **A / D**: Drive
- **SPACE**: Jump

## Goal
Destroy all the boxes in the level to unlock the next stage.
